About Me

My name is Kajogbola Faatih and I'm a passionate game developer and cybersecurity professional with expertise in creating immersive gaming experiences and protecting digital assets. I hold a Master's degree in Cybersecurity from the National College of Ireland, along with several certifications in Game Development. I specialize in building immersive, scalable gaming experiences and implementing robust security measures using my deep understanding of both game engines and security frameworks.


My game development journey spans multiple genres and platforms, from mobile games to more advanced PC experiences. I'm proficient in Unreal Engine Blueprint scripting and C++, with a strong focus on gameplay mechanics, optimization, and user experience. On the cybersecurity side, I focus on system hardening, penetration testing, and secure website and application development. I don't just build secure systems, I build systems that are fundamentally creative, robust, and designed to withstand real-world threats from the ground up.


I aim to deliver high-quality, immersive experiences where security is incorporated into every aspect of their design, not just as a mere afterthought. I'm always excited to take on new challenges and collaborate with like-minded individuals. Together, I believe we can build something truly exceptional and unbreakable.

Cloud Architecture Project

The main objective of this project was to set up a cloud environment and then completely secure it by initiating a Cloud VM Instance (IaaS) via AWS, configuring a LAMP Stack and a WordPress site on the same instance, and securing both the EC2 instance and the WordPress site. AI/ML Hybrid Deep Learning Framework

This project proposes a way to enhance intrusion detection capabilities using a hybrid deep learning model. The model will use an Autoencoder (AE) to learn normal network behaviour and an XGBoost classifier, using the AE's reconstruction error as a feature to identify anomalies. The Shapley Additive Explanations (SHAP) XAI technique will be incorporated to provide enhanced interpretability. The model’s performance was eveluated against simulated zero-day attacks using the trained CICIDS2017 dataset, which is a benchmark containing diverse network traffic flows including normal activity and common cyberattacks. The performance was then compared against baseline models, focusing on detection rates for unseen attacks and overall classification accuracy while maintaining low false positives.
